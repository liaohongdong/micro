# `@fe-micro/micro-util`

## npm script

``` shell
#启动服务
npm run serve

#打包编译 
npm run build

#执行所有测试用例
npm run test 

#执行单个测试用例(输入测试用例文件名即可，无需输入后缀)
npm run test ${yourfileName} 
```
